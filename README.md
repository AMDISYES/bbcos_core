# BBCOS Core
### 基于Python制作的“操作系统”
----
### 支持日历功能，时间功能，计算器功能
### 也有一些没用的功能：ls,coverter
----
### 使用方法

用户名: root

默认密码: 114514

```help```获取帮助

```calc```简易计算器

```ls```查看目录（没啥用）

```version```查看版本

```time```获取时间和日期

```calendar```日历

```passwd```修改密码

```clear```清屏

```coverter```文件转换器（又一个没用的）

```exit```注销
